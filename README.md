# Bicing – Base de dades
Aquest repositori conté el desenvolupament d’una base de dades per a la gestió del servei **Bicing**, a partir del disseny realitzat al Mòdul 1 del curs. 
En el Mòdul 2 s’està implementant aquest disseny mitjançant SQL.

La base de dades permet gestionar:
- Usuaris del servei
- Estacions
- Bicicletes
- Trajectes entre estacions
- Incidéncies detectades a les bicicletes
- Possible inserció de noves funcionalitats

Actualment el projecte inclou:
- El codi complet de la base de dades
- Inserció de dades bàsiques (usuaris, estacions i bicicletes)
- Desenvolupament de la inserció de trajectes
- **Preparació de consultes SQL segons el temari, en estat pendent**

El codi SQL està organitzat per fitxers per facilitar el manteniment i el treball col·laboratiu.
